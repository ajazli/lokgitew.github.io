# Lok Gitew Website

Singaporean & Malaysian Street Food website for Lok Gitew café.

## Folder Structure

```
lokgitew-website/
├── index.html          # Main HTML file
├── style.css           # All CSS styles
├── script.js           # JavaScript functionality
├── images/             # All images folder
│   ├── logo.png        # Lok Gitew logo (transparent background)
│   ├── loklok-platter.jpg
│   ├── rice-bowl-top.jpg
│   ├── oreo-shake.jpg
│   ├── loklok-sharing.jpg
│   └── rice-bowl-close.jpg
└── README.md           # This file
```

## Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Modern Street Food Aesthetic** - Bebas Neue & Poppins fonts
- **Brand Colors** - Bright orange (#FF6B35) and tangerine (#FF9F1C)
- **Smooth Animations** - Fade-in effects and hover interactions
- **Instagram Feed Integration** - Live feed from @lokgitew
- **Google Maps Integration** - Interactive location map
- **Social Media Links** - Instagram and TikTok

## Sections

1. **Hero** - Logo, tagline, and call-to-action
2. **About** - Story and concept
3. **Features** - What makes Lok Gitew special
4. **Gallery** - Food photography showcase
5. **Instagram Feed** - Live posts from @lokgitew
6. **Location** - Address, hours, contact info, and map
7. **Footer** - Quick links and social media

## How to Use

1. Extract all files maintaining the folder structure
2. Open `index.html` in a web browser to view locally
3. Upload all files to your web hosting server maintaining the same structure

## Customization

### Update Contact Information
Edit `index.html` in the Location section:
- Address
- Phone number
- Email
- Opening hours

### Update Instagram Feed
The Instagram feed is powered by Mirror App. The current embed code is already integrated.

### Change Colors
Edit `style.css` at the top where CSS variables are defined:
```css
:root {
    --primary-orange: #FF6B35;
    --tangerine: #FF9F1C;
    --bright-orange: #FF5722;
    --black: #1A1A1A;
    --white: #FFFFFF;
}
```

### Replace Images
Simply replace the image files in the `images/` folder with your own, keeping the same filenames.
